# Mi Portafolio

Portafolio personal profesional construido con React, TypeScript y Vite.

## Comandos

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Personalizacion

- Edita tu nombre, correo y enlaces en `src/App.tsx`.
- Reemplaza los proyectos de ejemplo por casos reales.
- Cambia la imagen principal en `src/assets/hero.png`.
- Ajusta colores y espaciado en `src/index.css` y `src/App.css`.
